# 使い方ガイド

このガイドでは、Googleカレンダー アポ候補自動抽出アプリの具体的な使い方を説明します。

## 前提条件：Googleカレンダーでの準備

このアプリを使用する前に、Googleカレンダーで以下のように予定を登録しておく必要があります。

### 1. アポ可能な時間枠の登録

アポイントが可能な時間帯に、**「アポ可能」**というキーワードを含む予定を作成します。

**例：**
- タイトル: 「アポ可能」
- 日時: 11月4日（木） 11:00-19:00

または、より詳細に：
- タイトル: 「営業アポ可能時間」
- 日時: 11月4日（木） 11:00-19:00

> 💡 ポイント: タイトルまたは説明欄に「アポ可能」という文字が含まれていればOKです

### 2. NG予定（埋まっている時間）の登録

すでに予定が入っている時間は、以下のマーカーを使って登録します：

#### 商談の予定
タイトルに **「★」** を含めます。

**例：**
- 「★A社商談」
- 「★新規顧客MTG」

#### インタビューの予定
タイトルに **「⚫︎」または「●」** を含めます。

**例：**
- 「⚫︎候補者面接」
- 「●採用インタビュー」

### 3. カレンダー登録例

以下のようにカレンダーに予定を登録すると：

```
11月4日（木）
11:00-19:00  アポ可能
13:00-14:00  ★A社商談
16:00-17:00  ⚫︎候補者面接

11月5日（金）
09:00-21:00  アポ可能
（他の予定なし）
```

アプリは以下のように抽出します：

```
・11/4 (木) 11:00~13:00、14:00~16:00、17:00~19:00
・11/5 (金) 終日
```

## アプリの使い方

### ステップ1: ログイン

1. アプリにアクセス
2. 「**Googleでログイン**」ボタンをクリック
3. Googleアカウントを選択
4. 権限の許可画面が表示されたら「**許可**」をクリック
   - カレンダーの読み取り権限のみを使用します

### ステップ2: 期間を指定

#### クイック選択を使う場合

- **「今週」ボタン**: 今週月曜日から日曜日までを自動選択
- **「来週」ボタン**: 来週月曜日から日曜日までを自動選択

#### カスタム期間を指定する場合

1. 「開始日」のカレンダーアイコンをクリック
2. 開始日を選択
3. 「終了日」のカレンダーアイコンをクリック
4. 終了日を選択

### ステップ3: 候補を抽出

1. 「**候補を抽出する**」ボタンをクリック
2. 数秒待つ（通常は3-5秒以内に完了）
3. 画面下部に結果が表示されます

### ステップ4: 結果をコピー

1. 結果表示エリアの右上にある「**結果をコピーする**」ボタンをクリック
2. クリップボードにコピー完了のメッセージが表示されます
3. メールやチャットアプリを開き、貼り付け（Ctrl+V または Cmd+V）

## 出力形式の詳細

### 基本形式

```
・月/日 (曜日) 時間
```

### パターン例

#### 1. 複数の時間帯がある場合
```
・11/4 (木) 11:00~13:00、14:00~16:00、17:00~19:00
```

#### 2. 午後のみ空いている場合
```
・11/4 (木) 14:00~18:00
```

#### 3. 夕方以降が空いている場合
```
・11/4 (木) 18:00以降
```

#### 4. 終日空いている場合
```
・11/5 (金) 終日
```

#### 5. 祝日の場合
```
・11/3 (月祝) 11:00~17:00
```

## 実際の使用例

### 例1: メールでの日程調整

**シチュエーション**: お客様に複数の候補日時を提示する

**操作手順**:
1. アプリで「来週」を選択
2. 「候補を抽出する」をクリック
3. 結果をコピー
4. メールに貼り付け

**メール文例**:
```
お世話になっております。

以下の日程でご都合はいかがでしょうか？

・11/4 (木) 11:00~15:00、16:00~18:00、19:00以降
・11/5 (金) 終日
・11/8 (月) 14:00~17:00

ご検討のほど、よろしくお願いいたします。
```

### 例2: Slackでの日程共有

**シチュエーション**: チーム内で空き時間を共有する

**操作手順**:
1. アプリで「今週」を選択
2. 結果をコピー
3. Slackに貼り付け

**Slackメッセージ例**:
```
今週のアポ可能時間です 📅

・11/2 (火) 15:00~18:00
・11/3 (水) 11:00~12:00、14:00~17:00
・11/4 (木) 終日
```

## よくある質問（FAQ）

### Q1: 「アポ可能」が複数のカレンダーにある場合は？

A: 現在のバージョンでは、プライマリカレンダー（メインのカレンダー）のみを対象としています。

### Q2: 「★」や「⚫︎」以外のマーカーも使えますか？

A: 現在は「★」と「⚫︎」（●も可）のみに対応しています。カスタマイズが必要な場合は、ソースコードを修正することで対応可能です。

### Q3: 過去の日付も抽出されますか？

A: 指定した期間内の予定がすべて抽出されます。通常は今日以降の日付を指定することをお勧めします。

### Q4: 同じ時間帯に「アポ可能」と「★商談」が重複している場合は？

A: 「★商談」の時間が優先され、その時間帯は候補から除外されます。

### Q5: 30分単位で細かく時間を指定したい場合は？

A: Googleカレンダー側で30分単位で予定を登録すれば、そのまま反映されます。

**例**:
```
11:00-11:30  アポ可能
11:30-12:00  ★商談
12:00-13:00  アポ可能

→ 抽出結果: 11:00~11:30、12:00~13:00
```

### Q6: ログインしたままにしておけますか？

A: はい、ログイン状態は維持されます。ただし、セキュリティの観点から定期的に再認証が必要になる場合があります。

### Q7: スマートフォンでも使えますか？

A: はい、レスポンシブデザインに対応しているため、スマートフォンでも快適に使用できます。

### Q8: 複数人のカレンダーをまとめて確認できますか？

A: 現在のバージョンでは対応していません。将来のアップデートで検討予定です。

## トラブルシューティング

### 予定が抽出されない

**確認事項**:
- [ ] カレンダーの予定に「アポ可能」というキーワードが含まれているか
- [ ] 指定した期間内に予定があるか
- [ ] Googleカレンダーの同期が完了しているか（予定を追加した直後など）

### 除外されるべき時間が含まれている

**確認事項**:
- [ ] NG予定のタイトルに「★」または「⚫︎」が含まれているか
- [ ] 全角文字で入力されているか（半角の「*」や「・」では認識されません）

### コピーボタンが動作しない

**確認事項**:
- [ ] ブラウザがクリップボードへのアクセスを許可しているか
- [ ] HTTPSで接続されているか（本番環境の場合）
